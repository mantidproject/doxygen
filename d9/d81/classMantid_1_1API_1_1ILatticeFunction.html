Detailed Description

ILatticeFunction.

This abstract class defines the interface for a function that calculates values based on HKL-values and a lattice. Currently the only implementation is CurveFitting::LatticeFunction.

Author
Michael Wedel, Paul Scherrer Institut - SINQ
