Detailed Description

template<typename TYPE = std::string>
class Mantid::Kernel::MaskedProperty< TYPE >

A property class for masking the properties.

It inherits from PropertyWithValue. This class masks the properties and useful when the property value is not to be displayed in the user interface widgets like algorithm history display, log window etc and log file .The masked property will be displayed. This class is specialised for string and masks the property value with charcater "*"
