Detailed Description

SCDCalibratePanels2 : Using input peakworkspace with indexation results to calibrate each individual panels.

The target calibration properties include:

  • T0: micro seconds time for proton to travel from reactor to target to generate neutron
  • L1: meters distance between target and sample
  • L2: meters (also known as z_shift) distance between sample and the center of each panel
  • Rot: degrees Euler angles (xyz )

Spirit successor of ISAW and its reincarnation: SCDCalibratePanels
