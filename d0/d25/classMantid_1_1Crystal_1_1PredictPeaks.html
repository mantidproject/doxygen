Detailed Description

Using a known crystal lattice and UB matrix, predict where single crystal peaks should be found in detector/TOF space.

Creates a PeaksWorkspace containing the peaks at the expected positions.

)
private

Calculates Q from HKL and adds a peak to the output workspace.

This method takes HKL and uses the oriented UB matrix (UB multiplied by the goniometer matrix) to calculate Q. It then creates a Peak-object using that Q-vector and the internally stored instrument. If the corresponding diffracted beam intersects with a detector, the peak is added to the output- workspace.
