Detailed Description

This is a specialization of IFunction for functions of one real argument.

It uses FunctionDomain1D as a domain. Implement function1D(...) method in a concrete function. Implement functionDeriv1D to use analytical derivatives.

Author
Roman Tolchenov, Tessella Support Services plc
Date
16/10/2009
