Detailed Description

The primitive types published by this API.

  • FLOAT32 32-bit float.
  • FLOAT64 64-bit double
  • INT8 byte-width integer – int8_t
  • UINT8 byte-width unsigned integer – uint8_t
  • INT16 double-byte-width integer – int16_t
  • UINT16 double-byte-width unsigned integer – uint16_t
  • INT32 quad-byte-width integer – int32_t
  • UINT32 quad-byte-width unsigned integer – uint32_t
  • INT64 eight-byte-width integer – int8_t if available on the machine
  • UINT64 eight-byte-width integer – uint8_t if available on the machine
  • BINARY lump of binary data, same as UINT8
